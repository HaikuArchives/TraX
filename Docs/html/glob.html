A good shell tutorial should be able to explain the use of globs to you.
BeForever has a Bash tutorial which might give you some help.
The quick tips are:

  • * matches any string
  • ? matches any character
  • [xy] matches an x or y
  • [a-zA-Z] matches any alphabetic character
eg *9907[01]*[0-9]? will match any filename of the form:
  • Any number of characters
  • Followed by '9907'
  • Followed by either a 0 or 1
  • Followed by any number of characters
  • Then ending in a digit, followed by another character
And yes, this sequence of characters actually has meaning to me.
